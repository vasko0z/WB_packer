# WB Packer

**Версия:** 1.0.1
**Последнее обновление:** 2026-06-02

WB Packer — приложение для управления и сборки поставок, разработанное для оптимизации процесса упаковки товаров по коробкам.

## Быстрый старт

- [Установка и запуск](docs/README.md)
- [Журнал изменений](SESSION_LOG.md)
- [Система документации](docs/PROJECT_SYSTEM.md)
- [CONTRIBUTING](CONTRIBUTING.md)

## Ключевые возможности

- Создание поставок из Excel/Google Sheets
- Групповые поставки с несколькими направлениями
- Управление коробками и сканирование штрихкодов
- Интеграция с МойСклад
- Многопользовательская работа через PostgreSQL
- Печать этикеток и экспорт отчётов

## Сборка

```bash
pyinstaller WB_Packer.spec --clean
```

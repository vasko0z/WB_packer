# WB Packer — Система сборки поставок

## Описание проекта

WB Packer — это приложение для управления и сборки поставок, разработанное для оптимизации процесса упаковки товаров по коробкам. Приложение позволяет создавать поставки на основе Excel-файлов, распределять товары по коробкам, отслеживать остатки и синхронизировать данные с системой МойСклад. 

Приложение поддерживает многопользовательскую работу с централизованной PostgreSQL базой данных, что позволяет нескольким сотрудникам работать с поставками одновременно. Система включает в себя функции архивации поставок, проверки остатков, печати этикеток и многое другое.

## Функции приложения

### Основные функции
- **Создание поставок**: Загрузка состава поставки из Excel-файла с автоматическим созданием новой поставки
- **Групповые поставки**: Поддержка создания групп поставок с несколькими направлениями из одного файла
- **Управление коробками**: Создание, переименование и удаление коробок, распределение товаров по коробкам
- **Сканирование штрихкодов**: Быстрое добавление товаров в коробки с помощью сканера штрихкодов
- **Добавление всего оставшегося**: Возможность добавить все оставшиеся товары в коробку одним кликом
- **Отслеживание прогресса**: Визуализация прогресса выполнения поставок с цветовой индикацией статуса
- **Многопользовательская работа**: Поддержка работы нескольких пользователей с отслеживанием активных пользователей

### Интеграция с МойСклад
- **Синхронизация остатков**: Получение актуальных остатков из МойСклад по указанным складам
- **Пакетная синхронизация**: Асинхронная синхронизация остатков для всех неархивных поставок
- **Реалтайм синхронизация**: Обновление остатков для текущей поставки в реальном времени
- **Настройка складов**: Возможность выбора складов для синхронизации в настройках пользователя
- **Кэширование остатков**: Локальное кэширование остатков для ускорения работы

### Управление поставками
- **Архивация поставок**: Возможность отправки поставок в архив для освобождения основного списка
- **Восстановление из архива**: Восстановление поставок из архива при необходимости
- **Обновление состава**: Возможность обновления состава поставки из нового Excel-файла
- **Удаление поставок**: Безопасное удаление поставок с подтверждением
- **Переименование поставок**: Изменение названия поставок и групп поставок

### Управление товарами
- **Удаление из поставки**: Возможность отметить товар как "удаленный из поставки" без полного удаления
- **Контроль остатков**: Проверка соответствия количества товаров в коробках и в поставке
- **Редактирование количества**: Возможность изменения количества товаров как в поставке, так и в коробках
- **Индикация проблем**: Визуальное выделение товаров с проблемами (превышение, удаленные и т.д.)

### Печать и отчеты
- **Печать этикеток**: Печать этикеток для товаров и коробок с настраиваемыми параметрами
- **Лист на паллет**: Генерация Word-документа с информацией о поставке для паллеты
- **Экспорт коробок**: Экспорт содержимого коробок в Excel-файл для передачи в логистику
- **Проверка поставки**: Сравнение фактического содержимого с ожидаемым по Excel-файлу

### Настройки и пользователи
- **Управление пользователями**: Создание и удаление пользователей системы
- **Персональные настройки**: Индивидуальные настройки шрифтов, тем, цветов и звуков для каждого пользователя
- **Настройки МойСклад**: Персональные настройки интеграции с МойСклад (токен, склады)
- **Темы оформления**: Поддержка светлой и темной тем с возможностью переключения

### Дополнительные функции
- **Автоматическое сохранение**: Отложенное сохранение изменений для улучшения производительности
- **Контроль сессий**: Отслеживание активности пользователей в поставках
- **Управление кэшем**: Эффективное кэширование данных для ускорения работы
- **Поддержка сетевых операций**: Оптимизация работы по сети с учетом задержек

### Интеграция с Google Sheets
- **Импорт SKU**: Актуализация таблицы SKU из Google Sheets (Штрихкод, Артикул, Наименование)
- **Импорт поставок**: Создание поставки из Google Sheets (кнопка "+ Поставка GH")
- **Нормализация штрихкодов**: Автоматическое удаление пробелов, дефисов, табуляций
- **Массовая вставка**: Использование `psycopg2.extras.execute_values` для быстрой вставки

## Установка и запуск

### Требования
- Python 3.8 или выше
- PostgreSQL сервер (для многопользовательской работы)
- Библиотеки, указанные в requirements.txt

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Настройка базы данных
1. Установите PostgreSQL сервер
2. Настройте параметры подключения в файле config.py:
   - `POSTGRESQL_HOST`: адрес сервера
   - `POSTGRESQL_PORT`: порт
   - `POSTGRESQL_DATABASE`: имя базы данных
   - `POSTGRESQL_USER`: имя пользователя
   - `POSTGRESQL_PASSWORD`: пароль

### Запуск приложения
```bash
python main.py
```

### Сборка исполняемого файла
```bash
pyinstaller WB_Packer.spec
```
